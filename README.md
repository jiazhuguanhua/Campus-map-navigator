# 南京工业大学非官方校园网站

## 功能介绍
这是一个生活服务导航页面，为南京工业大学学生提供便民服务导航，包含7个固定的服务地点导航按钮。

## 文件结构
```
二维码跳转/
├── index.html              # 主页面文件
├── styles.css              # 样式文件
├── script.js               # 功能脚本
├── resources/              # 资源文件夹
│   ├── background.png      # 默认背景图片
│   └── map.png            # 电子地图文件
└── README.md              # 使用说明（本文件）
```

## 页面特色
- **大学标识**: 明确标注"南京工业大学非官方校园网站"
- **欢迎横幅**: 页面顶部显示"欢迎新同学"
- **电子地图下载**: 醒目的"获取电子地图"按钮，点击下载校园地图
- **默认背景**: 使用 `resources/background.png` 作为默认背景图片
- **优雅降级**: 如果背景图片不存在，自动使用渐变背景

## 主要功能

### 1. 导航按钮
- 7个预设服务地点按钮
- 点击按钮跳转到百度地图对应位置
- 响应式设计，支持手机和电脑

### 2. 电子地图下载
- **醒目下载按钮**: 在欢迎横幅中显示"获取电子地图"按钮
- **一键下载**: 点击按钮直接下载校园电子地图
- **自动命名**: 下载文件自动命名为"校园电子地图.png"

### 3. 交互体验
- 按钮点击动画效果
- 触摸反馈支持
- 消息提示系统
- 统计功能集成

### 4. 数据统计
- **访问量统计**: 集成百度统计，实时监控网站访问量
- **下载量统计**: 自动记录电子地图下载次数
- **用户行为分析**: 追踪按钮点击、背景更换等用户操作
- **本地备份统计**: 浏览器本地存储统计数据作为备份
- **实时数据**: 20分钟后可在百度统计后台查看详细数据

## 使用方法

### 项目部署
1. 将背景图片重命名为 `background.png` 并放置在 `resources/` 文件夹下
2. 将电子地图文件重命名为 `map.png` 并放置在 `resources/` 文件夹下
3. 直接打开 `index.html` 文件或部署到网站服务器

### 基本使用
1. 访问网站首页
2. 点击"获取电子地图"下载校园地图
3. 点击任意导航按钮跳转到对应地点
1. 打开 `index.html` 文件
2. 点击任意按钮导航到对应地点

### 自定义背景
1. **方法一**: 点击"更换背景"按钮，选择图片文件
2. **方法二**: 直接拖拽图片文件到页面任意位置
3. **恢复默认**: 点击"恢复默认"按钮

### 快捷键
- `Ctrl + Shift + S`: 查看本地统计数据（开发者功能）

## 统计功能

### 百度统计集成
本网站已集成百度统计服务，可以追踪以下数据：

#### 访问量数据
- **页面浏览量(PV)**: 用户访问页面的总次数
- **独立访客数(UV)**: 访问网站的不重复用户数
- **访问次数**: 用户访问网站的次数
- **平均访问时长**: 用户在网站停留的平均时间
- **跳出率**: 只浏览一个页面就离开的访问比例

#### 用户行为数据
- **地图下载次数**: 统计"获取电子地图"按钮的点击次数
- **导航按钮点击**: 记录每个服务地点按钮的点击情况
  - 阳光健身点击量
  - 久昌百货点击量
  - 魔世电竞点击量
  - 37℃干洗店点击量
  - 宁星驾校点击量
  - 雅迪电动车点击量
  - 博约公考点击量
- **快捷键使用**: 记录键盘快捷键的使用频率

#### 技术信息
- **浏览器类型**: 用户使用的浏览器分布
- **操作系统**: 访问用户的操作系统类型
- **屏幕分辨率**: 用户设备的屏幕尺寸
- **访问来源**: 用户从哪个网站/搜索引擎进入

### 数据查看方式

#### 百度统计后台
1. 访问 [百度统计](https://tongji.baidu.com/)
2. 使用配置的账号登录
3. 查看详细的访问量和用户行为数据
4. 数据更新：代码安装正确后，一般20分钟可查看数据

#### 本地统计查看
- 按 `Ctrl + Shift + S` 查看本地统计数据
- 包含下载次数、访问记录等基础信息
- 作为百度统计的补充和备份

### 统计代码说明
- 使用百度统计最新版异步加载代码
- 适用于PC端和移动端
- 不影响网站页面显示和加载速度
- 数据准确性高，加载速度快

## 支持的图片格式
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- BMP (.bmp)

## 注意事项
1. 背景图片会保存到浏览器本地存储
2. 过大的图片可能无法保存，建议使用小于5MB的图片
3. 清除浏览器数据会丢失自定义背景
4. 建议使用横向比例的图片作为背景
